# AnimeWorld v2

## Setup

### Frontend (client/)
```
cd client
npm install
cp .env.example .env
# Fill in .env with your values
npm run dev
```

### Backend (server/)
```
cd server
npm install
cp .env.example .env
# Fill in .env with your values
npm start
```

## Environment Variables
See `.env.example` in each folder for required variables.
Never commit real .env files.
